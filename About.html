Implementation Principles

Responsiveness

Responsive web design is a process to create a webpage that uses flexible layouts, images and cascading styles sheets queries to detect the users’ screen size, orientation and changes the layout accordingly. Regardless of the device, the webpage should be easy to use and make it look good on any screen.
